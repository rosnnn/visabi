# Success Stories case study images (optional)

Case study pages currently use images from Unsplash (configured in `src/data/successStoryCaseStudies.js`).

To use your own images instead, add files here and update the `images` object in that file, for example:

- **Entellus:** security, fire prevention, guards (e.g. `entellus-hero.jpg`, `entellus-section1.jpg`, `entellus-section2.jpg`)
- **Global hi-tech:** aircraft, aerospace, defense (e.g. `global-hero.jpg`, `global-section1.jpg`, `global-section2.jpg`)
- **Vector:** office, team, HR (e.g. `vector-hero.jpg`, `vector-section1.jpg`, `vector-section2.jpg`)

Then set each URL to `"/images/success-stories/your-filename.jpg"` in `successStoryCaseStudies.js`.
